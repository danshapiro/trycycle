# Trycycle

This repo is a skill for AI coding agents (Claude Code, Codex CLI, Gemini CLI).
Users install it by cloning into their skills directory. Do not modify files
unless the user explicitly asks.

## Structure

- `SKILL.md` + `skills/` — the skill itself (what users install and run)
- `maintenance/` — scripts for updating skills from upstream (not user-facing)
- `README.md` — human-facing install and usage docs

## If you are here to install trycycle

Follow the instructions in README.md. Just clone — no other dependencies needed.

## If you are here to work on trycycle

The skills in `skills/` are generated by `maintenance/import-skills.sh` from
upstream superpowers (by Jesse Vincent). Edit the prompt template and
skill-specific instructions in `maintenance/`, then re-run the import script.
Do not hand-edit the generated skill files.
